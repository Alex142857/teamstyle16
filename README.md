## 关于Git
请大家 ***不要*** 使用中文文件名或目录名 = =

请大家 ***不要*** 直接对 **master** 分支进行修改，而是将 **develop** 分支作为主要开发分支。（为了保险起见我还是把 **master** 设为protected了…Orz）

请大家 ***不要commit写到一半的代码***，对每次commit***附上清晰简明的概要***，并且***在本地测试没有问题之后再push***

请大家在merge时***使用*** `--no-ff` ***参数***，以保持branch层次的清晰
